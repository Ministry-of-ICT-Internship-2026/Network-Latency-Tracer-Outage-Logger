# Network Latency Tracer & Outage Logger

## Overview

This project is a Python-based application that monitors network latency, detects outages, stores network logs in SQLite, and displays real-time statistics through a Streamlit dashboard.

## Features

- Monitor network latency
- Detect network outages
- Store historical logs
- Display a live dashboard
- Generate reports

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Plotly

## How to Run

1. Install Python.
2. Install the project dependencies:

```bash
pip install -r requirements.txt
```

3. Start the application:

```bash
streamlit run app.py
```
